# Änderungen online stellen

1. Änderung mit Claude machen
2. Terminal öffnen und in den Projektordner navigieren:

```
cd ~/Desktop/KI-System-Onepager
```

3. Hochladen:

```
git add . && git commit -m "Beschreibung der Änderung" && git push
```

Fertig — Seite ist nach ~10 Sekunden live unter:
https://ki-system-onepager.vercel.app
