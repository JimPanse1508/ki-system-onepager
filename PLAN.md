# KI-System Onepager – Projektplan

## Ziel
Ein professioneller, verkaufsfähiger One-Pager der mein KI-Dienstleistungsangebot präsentiert.
Gebaut mit Claude Code – gleichzeitig als Beweis meiner eigenen KI-Skills gegenüber Kunden.

---

## Tech Stack
- Reines HTML / CSS / Vanilla JS (keine Frameworks)
- Google Fonts: Syne (Display) + DM Mono (Labels)
- Keine externen Dependencies
